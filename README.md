# Budapest Flat Sale Website

A simple, professional website to showcase your flat for sale in Budapest.

## Setup Instructions

1. **Add Your Property Details**
   - Open `index.html`
   - Replace all placeholders marked with `[BRACKETS]` with your actual information:
     - Size, bedrooms, bathrooms
     - Location/district
     - Floor number
     - Price
     - Address
     - Contact information

2. **Add Photos**
   - Create an `images` folder in this directory
   - Add your property photos with these names:
     - `main.jpg` - Main hero image
     - `living-room.jpg`
     - `kitchen.jpg`
     - `bedroom.jpg`
     - `bathroom.jpg`
   - Or update the image paths in `index.html` to match your photo names

3. **Customize Features**
   - Edit the features list in `index.html` to match your flat's actual amenities
   - Add or remove items as needed

4. **Add Google Maps**
   - Go to [Google Maps](https://www.google.com/maps)
   - Search for your address
   - Click "Share" → "Embed a map"
   - Copy the iframe code
   - Replace the map placeholder in `index.html` with the iframe

5. **Test Locally**
   - Open `index.html` in your web browser
   - Check that everything looks correct

## Deployment Options

### Option 1: GitHub Pages (Free)
1. Create a GitHub account if you don't have one
2. Create a new repository
3. Upload these files
4. Go to Settings → Pages
5. Enable GitHub Pages
6. Your site will be live at `username.github.io/repository-name`

### Option 2: Netlify (Free)
1. Go to [netlify.com](https://www.netlify.com)
2. Drag and drop this folder
3. Your site will be live instantly

### Option 3: Any Web Hosting
- Upload all files to your web hosting via FTP
- Access via your domain

## Features

- Responsive design (works on mobile, tablet, desktop)
- Image gallery with clickable thumbnails
- Property details section
- Contact form
- Modern, professional appearance

## Customization

- **Colors**: Edit the gradient colors in `style.css` (search for `#667eea` and `#764ba2`)
- **Layout**: Modify the HTML structure in `index.html`
- **Fonts**: Change the font-family in `style.css`

## Contact Form

The contact form currently shows an alert. To make it functional:
- Use a service like [Formspree](https://formspree.io) (free)
- Or add backend code to send emails
- Or replace with a direct mailto link

## Tips for Better Results

1. Use high-quality photos (bright, well-lit)
2. Take photos from multiple angles
3. Clean and declutter before photographing
4. Highlight unique features of your flat
5. Be honest about the condition and details
6. Include nearby amenities and transport options
7. Update regularly if price or details change

## Legal Note

Make sure to comply with local real estate advertising regulations in Hungary.
